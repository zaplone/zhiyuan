# 启动说明（前端）

本仓库为**独立站前端**（`safetyshoe-frontend`）。业务数据来自独立站 API，生产基地址：

**`https://luyuan.zwstone.cn/api`**

## 本地运行

```bash
cd safetyshoe-frontend
npm install
npm run dev
```

浏览器访问：<http://localhost:3000>

## 环境变量（可选，`.env.local`）

```bash
NEXT_PUBLIC_SITE_API_URL=https://luyuan.zwstone.cn/api
```

未设置时，代码内默认与上式一致；若仍使用旧变量名 `NEXT_PUBLIC_STRAPI_URL`，也会被兼容读取。

## 生产构建

```bash
cd safetyshoe-frontend
npm run build
```

产物在 `out/`，供 Cloudflare Pages 等静态托管。
