# 2026-04-02 日志

## FactoryVideoGallery 工厂图片更换与翻译修复

### 问题
1. 关于页工厂展示图片（`/images/about/生产环境1-6.jpg`）使用的是旧占位图，且顺序与实际图片命名不符（质检区和装配区互换）
2. 图片上的文字（title/titleZh/description）硬编码在组件里，没有走 i18n

### 解决方案
1. 从 `网站材料/关于工厂/` 复制 6 张真实工厂图片到 `public/images/about/`：
   - 裁切区.jpg、车缝区.jpg、成型区.jpg、装配区.jpg、质检区.jpg、包装发货.png
2. 修改 `FactoryVideoGallery.tsx`：
   - FactoryZone 接口移除 title/titleZh/description，改为 titleKey（i18n key）
   - 图片路径更新为新图片，顺序：裁切→车缝→成型→装配→质检→包装发货
   - 渲染时通过 `t(\`zones.xxx.title\`)` 等 key 取翻译
3. 在 `en.json` 和 `zh.json` 的 `About` 节点下新增 `zones` 对象，包含每个区域的 title/zh/desc
