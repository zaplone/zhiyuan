# Luyuan Safety Shoes Frontend

Professional industrial safety footwear manufacturer website.

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- next-intl (i18n)
- Data API: `https://luyuan.zwstone.cn/api` (see `src/lib/siteApi.ts`)

## Deployment
Cloudflare Pages (static export `out/`). Set `NEXT_PUBLIC_SITE_API_URL` in Cloudflare environment variables.
